If you are external to Tufts and interested in using our Nf-core OpenOn Demand applications you are probably interested in 
the https://github.com/TuftsRT/nf-core_open-ondemand repository which allows you to create your site specific version of this.
This repository is the output of Tufts usage of that tool.  It is shared publicy as a reference implemenation.  You are welcome to use it, but it is better to create your own.
